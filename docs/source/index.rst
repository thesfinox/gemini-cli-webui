Welcome to Gemini Web UI's documentation!
=========================================

**Gemini Web UI** is a lightweight, elegant, and powerful web interface for Google's Gemini models, built on top of the robust ``gemini-cli``.

Powered by `Streamlit <https://streamlit.io/>`_, this application provides a user-friendly chat interface that bridges the gap between the command line and a full graphical experience.
